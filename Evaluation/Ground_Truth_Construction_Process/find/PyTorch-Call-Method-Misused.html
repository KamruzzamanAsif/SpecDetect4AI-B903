Exhaustive Checking Steps for PyTorch Call Method Misused

  1. Check if torch is imported.

  2. If PyTorch is imported:

    2.1 Search for calls to:

    • .forward(...) directly on a nn.Module subclass (e.g., self.net.forward(...), net.forward(...))

  3. If .forward(...) is used instead of calling the model directly (e.g., net(...)):

  • Mark as PyTorch Call Method Misused.
  1. If none of these patterns are found, or if PyTorch is not imported:
  • Skip this check.
